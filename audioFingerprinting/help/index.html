Audio Fingerprinting toolbox

Version 1.5, for MATLAB 8.5.0.197613 (R2015a)

  • Audio Fingerprinting
    • afpDbCreate: Create a database for AFP from a given audio directory
    • afpFea2hash: Convert landmarks to hash entries for storing
    • afpFeaExtract: Extract landmarks from audio file for audio fingerprinting
    • afpHashHitGet: Return hash values in a hash table given a set of given landmarks
    • afpOptSet: Set the options for AFP
    • afpOptSet0: Set the basic options for AFP
    • afpPerfEval: AFP performance evaluation
    • afpQueryMatch: Match a given query to an AFP database
  • Utility
    • afpEnvelopeGen: Generate the envelope for a given signal for AFP
    • afptRoot: Root of AFPT (Audio Fingerprinting Toolbox)
    • toolboxInfo: Toolbox information
